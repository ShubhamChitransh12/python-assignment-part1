# Student Grade Tracker

## Python Basics & Control Flow Project

## Overview
This project is a command-line based Student Grade Tracker built using core Python concepts. It demonstrates data cleaning, analysis and basic reporting using fundamental programming constructs.

## Tasks Covered

### Task 1: Data Cleaning & Profile Generation
- Clean and structure student data
- Validate names and format profiles

### Task 2: Marks Analysis
- Assign grades
- Calculate total, average, highest, lowest marks
- Add subjects using loop with input validation

### Task 3: Class Performance Summary
- Compute student averages and pass/fail status
- Generate class report and identify topper

### Task 4: String Manipulation Utility
- Clean and process essay text
- Perform text transformations and analysis


## Concepts Used 
- Loops and conditionals
- String operations
- Lists and dictionaries
- Input validation and exception handling

Run the Python file.
